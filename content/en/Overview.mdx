# Welcome

Hello and welcome! This is a Python learning companion website for our students. There are two main sections of this website that you access by clicking the cards below or the links in the navigation bar at the top of the page:
- Learn: Follow the course units in order that topics are taught in class.
- Reference: Look up Python concepts, syntax, and examples. Organized more conceptually for ease of reference.

<Callout type="info" title="Note about content">
  Some of the pages are still a work in progress. If you find any issues or have any suggestions, please let us know.

  Also by design, the Learn section will not contain all of the content that is in the Reference section.
  This is because we want to keep the Learn section focused on the most important content, while Reference is more comprehensive.
</Callout>
