# Project 2: Grade Tracker

You'll build a program that stores grades in a list, computes stats (average, highest, lowest), and later saves and loads grades from a file. The project is split into three phases that match the topics you learn each day.

---

### Learning focus
Create a list, loop through the list to display and to compute a sum.

**Topics covered:** [Lists](../../Variables/TypesOfVariables/DataStructures/Lists), [Loops](../../ControlFlow/Loops)

**End of Day 1:** You have a program with a list of grades that prints them and shows the sum and average.

### Challenge (optional)
Instead of a static list, add grades using a loop. Try using a `while` loop with a quit keyword — see [Stopping on a specific input](/en/Reference/ControlFlow/Loops/WhileLoops#stopping-on-a-specific-input).

<Details>
  <summary>Hint — what each function looks like</summary>

  ```
  function calculate_average:
      takes: a list of grades
      add up all the grades
      divide by the number of grades
      return the result

  function find_highest:
      takes: a list of grades
      keep track of the highest grade seen so far
      loop through the list, updating highest when you find a bigger value
      return highest

  function find_lowest:
      takes: a list of grades
      same idea as find_highest, but track the smallest value instead

  function print_stats:
      takes: a list of grades
      call calculate_average, find_highest, find_lowest
      print each result
      use if/elif/else to print a status message based on the average

  function load_grades:
      takes: a filename
      open the file for reading
      for each line, convert it to a float and add it to a list
      return the list

  function save_grades:
      takes: a list of grades and a filename
      open the file for writing
      write each grade on its own line
  ```
</Details>

<Details>
  <summary>Hint — what the main section looks like</summary>

  ```
  grades = load_grades("grades.txt")
  print_stats(grades)
  save_grades(grades, "grades.txt")
  ```
</Details>
