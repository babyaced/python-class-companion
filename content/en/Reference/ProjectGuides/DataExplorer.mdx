# Project 2: Data Explorer

You'll build a program that loads movie data from a file, analyzes it (average rating, highest-rated, filter by genre), and saves the results. This project uses **lists**, **loops**, **functions**, and **file handling** only — no dicts, classes, or objects.

**Representing movie data with lists:** Use a list of lists — each movie is a list with a fixed order, e.g. `[title, rating, genre]`. So `movies` might look like: `[["Inception", 8.8, "Sci-Fi"], ["Comedy Movie", 7.5, "Comedy"], ...]`. Use index `0` for title, `1` for rating, `2` for genre.
