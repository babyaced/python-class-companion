# Project 2: Data Explorer

You'll build a movie analyzer. Your program loads a list of movies from a file, runs some analysis — average rating, highest-rated film, filter by genre — and saves the results. You'll practice **lists**, **loops**, **functions**, and **file handling**.

**How you'll store movie data:** Each movie is a list with three values in a fixed order: `[title, rating, genre]`. Your full collection is a list of those lists. So `movies` might look like `[["Inception", 8.8, "Sci-Fi"], ["Clueless", 7.1, "Comedy"]]`. To get the title, use index `0`. Rating is index `1`, genre is index `2`.
