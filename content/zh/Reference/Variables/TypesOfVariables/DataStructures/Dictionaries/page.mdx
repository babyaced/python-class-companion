# 词典

想想真正的字典--每个词都有一个定义。在 Python 中，** 字典** 以同样的方式工作：它将数据存储为**键值对**，其中每个键映射到一个值。

当您的数据有标签时，您就会使用字典。字典可以让你按名称查找事物，而不是按位置追踪事物的列表。


```python
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}
```

这里，`"姓名"、`"年龄 "和`"城市 "是**键**。Alice"、"30 "和 "纽约 "是它们的**值。

## 创建字典

将键值对放在大括号`{}`内。用逗号分隔每个键值对，并在每个键和值之间使用冒号 `:`。

键值必须是唯一的，如果两次使用相同的键值，则第二个值会覆盖第一个值：


```python
person = {
    "name": "Alice",
    "name": "Aly"   # overwrites "Alice"
}
print(person)  # Output: {'name': 'Aly'}
```

值可以是任何类型--字符串、数字、布尔值、列表，甚至其他字典。
