# 访问值

## 使用方括号

要获取一个值，请在方括号中写入字典名和键：


```python
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}
print(person["age"])   # Output: 30
print(person["name"])  # Output: Alice
```

## 使用 get()

<get()` <GlossaryTerm term="method">方法</GlossaryTerm>做同样的事情，但如果键不存在，你可以提供一个后备值：


```python
print(person.get("age"))                        # Output: 30
print(person.get("occupation", "Not specified")) # Output: Not specified
```

如果在不存在的键上使用方括号，Python 会引发错误。`get()` 返回 `None` (或您的后备函数) - 当您不确定键是否存在时，这样更安全。

<Callout type="info" title="何时使用 get()">
当某个键可能不在字典中时，使用 `get()`。当您确定键存在，并希望 Python 在它不存在时引发错误时，使用方括号。
