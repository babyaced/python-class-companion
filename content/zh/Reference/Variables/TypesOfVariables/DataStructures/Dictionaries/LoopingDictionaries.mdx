# 循环字典

在字典中循环时，默认情况下只获取键值。

## 在键上循环


```python
person = {"name": "Alice", "age": 16, "city": "San Francisco"}

for key in person:
    print(key)
# name
# age
# city
```

您可以使用键来访问循环中的每个值：


```python
for key in person:
    print(key, ":", person[key])
# name : Alice
# age : 16
# city : San Francisco
```

## 循环数值

如果只需要数值，请使用 `.values()`：


```python
for value in person.values():
    print(value)
# Alice
# 16
# San Francisco
```

## 对键和值一起循环

使用 `.items()` 一次获得两个值。每对数据都会返回一个元组 `(key,value)`：


```python
for key, value in person.items():
    print(key, ":", value)
# name : Alice
# age : 16
# city : San Francisco
```

<Callout type="info">
  当您需要键和值时，".items() "是循环字典的最常用方法。
</Callout>
