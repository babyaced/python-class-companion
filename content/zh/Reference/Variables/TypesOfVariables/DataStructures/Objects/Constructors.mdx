# 构造函数

## 什么是构造函数？
构造函数**是一个特殊函数，当从<GlossaryTerm>类</GlossaryTerm>中创建<GlossaryTerm>对象</GlossaryTerm>时调用该函数。

我们经常使用构造函数来初始化对象的独特属性。

为了创建每个对象独有的属性，我们使用**实例变量**。

对于一只狗，我们可以使用**实例变量**来创建它的 "名字 "和 "年龄"。

要设置这些独特的属性，我们需要使用一种叫做**构造函数**的东西。

## 定义构造函数
在 Python 中，我们必须通过定义 `__init__` <GlossaryTerm> 方法</GlossaryTerm>来定义构造函数。

下面是一个如何使用构造函数创建具有唯一的 `name` 和 `age` 属性的 `Dog` 类的示例：


```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

在本例中，我们通过定义 `__init__` 方法创建了一个构造函数，该方法需要两个参数：名称 "和 "年龄"。

`self` 参数是对类的当前 <GlossaryTerm> 实例</GlossaryTerm> 的引用，用于访问属于类的变量。

现在，我们可以创建多个具有不同名称和年龄的狗对象：


```python
dog1 = Dog("Buddy", 3)
dog2 = Dog("Max", 5)
```

要使用构造函数，我们只需像调用其他函数一样，在类名后面写上括号，并传递所需的**参数**。

dog1 "和 "dog2 "是两个独立的实例--每个实例都有自己的 "名称 "和 "年龄"。


### 更多关于 `self` 的内容
一开始，`self` 的概念可能有点难以理解。
因此，让我们分解一下构造函数中发生的事情。

当我们使用 `Dog("Buddy", 3)` 创建一个新的狗对象时、
调用 `__init__` 方法时，"self "指的是正在创建的新的狗对象，也就是它的 "self"。

这个新 Dog 对象的 `name` 被设置为 "Buddy"，`age` 被设置为 3。
