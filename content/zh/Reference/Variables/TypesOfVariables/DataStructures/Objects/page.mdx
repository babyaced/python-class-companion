# 类和对象

你一直在使用对象。字符串、列表，甚至 Pillow 中的 `img` 和 `draw` 变量都是对象。现在你将了解这意味着什么--以及如何构建自己的对象。

### 什么是对象？

<GlossaryTerm>object</GlossaryTerm> 是一个变量，它将**数据**（属性）和**行为**（方法）捆绑在一起。

例如，列表对象存储条目（数据），并具有 `.append()` 和 `.pop()`（行为）等方法。

### 什么是类？

<GlossaryTerm>类</GlossaryTerm>是创建对象的蓝图。它定义了该类型的每个对象将拥有的属性和方法。

下面是一个表示狗的简单类：


```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

狗 "是蓝图。`self.name` 和 `self.age` 是属性，每只狗都有自己的名字和年龄。

## 创建对象

要从一个类中创建对象，可以像调用函数一样调用该类：


```python
dog1 = Dog("Buddy", 3)
dog2 = Dog("Max", 5)
```

每个对象都被称为类的**实例**。dog1 "和 "dog2 "是两个独立的实例--它们共享相同的蓝图，但名字和年龄不同。

## 访问属性

使用**点符号**访问对象的属性：


```python
print(dog1.name)  # Buddy
print(dog2.age)   # 5
```

<Callout type="info">
  你以前使用过点符号--`my_list.append()`、`"hello".upper()`。这些都是对列表和字符串对象的方法调用。
</Callout>

## 类变量

有些属性对每个实例都是相同的。你可以直接在类上定义这些属性--在 `__init__` 之外--它们将在所有对象中共享：


```python
class Dog:
    species = "Canis familiaris"

    def __init__(self, name, age):
        self.name = name
        self.age = age

dog1 = Dog("Buddy", 3)
dog2 = Dog("Max", 5)

print(dog1.species)  # Canis familiaris
print(dog2.species)  # Canis familiaris
```

种类 "是一个**类变量**--每只狗都共享它。名字 "和 "年龄 "是**实例变量**，每只狗都有自己的名字和年龄。
