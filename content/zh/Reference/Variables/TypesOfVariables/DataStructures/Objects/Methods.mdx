# 方法

<GlossaryTerm>method</GlossaryTerm> 是属于一个类的函数。方法定义了对象可以***做的事情。

## 定义方法

方法是在类中定义的，就像普通函数一样，但它们总是将 `self` 作为第一个参数。


```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print("Woof!")

    def introduce(self):
        print("Hi, I'm " + self.name)
```

self` 为方法提供了访问对象属性的权限。没有它，`introduce` 就不知道该打印哪只狗的名字。

## 调用方法

使用点符号调用对象上的方法：


```python
dog1 = Dog("Buddy", 3)
dog1.bark()       # Woof!
dog1.introduce()  # Hi, I'm Buddy
```

调用时不需要传递 `self` - Python 会自动处理。

## 带参数的方法

除了 `self` 之外，方法还可以接受额外的参数：


```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def eat(self, food):
        print(self.name + " is eating " + food)
```


```python
dog1 = Dog("Buddy", 3)
dog1.eat("kibble")  # Buddy is eating kibble
```

<Callout type="info">
  这正是 Pillow 的工作原理--`draw.ellipse(...)` 正在调用`draw`对象上的`ellipse`方法，并将边界框和颜色作为参数传入。
</Callout>

## 方法可以使用其他方法

方法可以使用 `self` 在同一对象上调用其他方法：


```python
class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print("Woof!")

    def greet(self):
        print("Hi, I'm " + self.name)
        self.bark()
```


```python
dog1 = Dog("Buddy")
dog1.greet()
# Hi, I'm Buddy
# Woof!
```
