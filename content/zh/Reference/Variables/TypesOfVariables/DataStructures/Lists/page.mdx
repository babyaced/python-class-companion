# 列表

列表可让你在一个地方按顺序存储多个项目，比如歌曲播放列表、一组测验分数或待办事项列表。你可以添加、更改和删除项目，每个项目都有一个称为 **index** 的位置。

列表中的项目可以是数字、字符串、布尔值，甚至是其他列表。大多数情况下，您会为每个列表保留一种类型（所有字符串或所有数字），这样您的代码就更容易处理了。

## 创建列表

将项目放在方括号"[]"内，用逗号分隔：


```python
numbers = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "cherry"]
```

### 空列表

您可以创建一个空列表，然后再填写：


```python
scores = []
```

这在您还不知道项目时非常方便--例如，如果您在一个循环中收集用户输入，并在进行过程中添加每个条目。
