# 列表方法

列表内置了**方法**--可以在列表上运行的操作。调用时，先写入列表名、点号，然后写入方法名：`my_list.method()`。

下面是你最常用的方法：

## append()

在列表末尾添加一个项目：


```python
fruits = ["apple", "banana"]
fruits.append("cherry")
print(fruits)  # Output: ['apple', 'banana', 'cherry']
```

## remove()

移除列表中与所传值匹配的第一个项目：


```python
fruits = ["apple", "banana", "cherry"]
fruits.remove("banana")
print(fruits)  # Output: ['apple', 'cherry']
```

## pop()

移除一个项目并将其还给你。如果没有参数，则会删除最后一个项目：


```python
fruits = ["apple", "banana", "cherry"]
last = fruits.pop()
print(last)    # Output: cherry
print(fruits)  # Output: ['apple', 'banana']
```

## insert()

在特定索引处插入一个项目。该位置及其后的所有内容都会右移以腾出空间：


```python
fruits = ["apple", "banana"]
fruits.insert(1, "cherry")
print(fruits)  # Output: ['apple', 'cherry', 'banana']
```

## sort()

按升序（从小到大，或从 A 到 Z）对列表排序：


```python
numbers = [3, 1, 4, 2]
numbers.sort()
print(numbers)  # Output: [1, 2, 3, 4]
```

要反向排序，请输入 `reverse=True`：


```python
numbers.sort(reverse=True)
print(numbers)  # Output: [4, 3, 2, 1]
```

## len()

返回列表中有多少个项目。请注意，`len()` 是一个内置函数，而不是列表方法，因此不能使用点符号：


```python
fruits = ["apple", "banana", "cherry"]
print(len(fruits))  # Output: 3
```
