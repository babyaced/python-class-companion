# 什么是条件式？

**条件***让你的程序做出决定：当某些事情为真时运行一个代码块，当它为假时运行另一个代码块。在 Python 中，您可以使用**比较运算符** (如 `==`, `<`, `>=`) 和**逻辑运算符** (`and`, `or`, `not`)来建立条件，然后在**`if`语句**中使用它们（之后在循环中使用）。

## 你将在这里学到什么

- 比较运算符** - 比较两个值，得到`真`或`假`（`==`, `!=`, `<`, `>`, `<=`, `>=`）。
- 逻辑运算符** - 用`and`、`or`和`not`组合条件。
- **If 语句** - 仅在条件为真时运行代码，并使用 `else` 和 `elif` 来处理多种情况。

按顺序完成这些工作：比较和逻辑运算符提供了构建模块；if 语句是使用它们的地方。
