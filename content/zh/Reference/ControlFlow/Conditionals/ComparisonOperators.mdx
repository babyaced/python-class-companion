# 比较运算符

您将学习六个**比较运算符**--它们比较两个值，并给出 "真 "或 "假"。您可以在**条件**（如在`if`或`while`循环中）中使用它们，这样您的程序就可以决定是否运行某个代码块。

## 六个比较运算符

每个运算符在两个值之间进行比较。结果总是 `True` 或 `False`（布尔值）。

| 操作符 | 含义 | 示例 | 结果
|----------|-------------------|---------------|----------|
| `==` | 等于 | `5 == 5` | `True` | `!
| `!=` | 不等于 | `5 != 3` | `True` |
| `<` | 小于 | `3 < 5` | `True` |
| `>` | 大于 | `5 > 3` | `True` | `<=` | `5 > 3` | `True` |
| `<=` | 小于或等于 | `3 <= 3` | `True` |
| `>=` | 大于或等于 | `5 >= 5` | `True` |

要检查两个值是否相等，请使用 ** 两个**等号：`==`.单个 `=` 为 <GlossaryTerm term="variable"> 变量</GlossaryTerm>赋值；它不进行比较。
</Callout> 呼出

## 在条件中使用比较操作符

您通常会在 `if`（或 `while`）中使用比较操作符。条件要么为真，要么为假，Python 只在条件为真时运行代码块。


```python
age = 15

if age >= 13:
    print("You can see this movie.")
else:
    print("This movie is for older audiences.")
```

这里，`年龄 >= 13` 是条件。Python 将其评估为 `True` 或 `False`，然后选择运行哪个代码块。您可以比较数字、字符串 (例如，`name == "Alex"`)，以及其他类型--只需使用有意义的值进行比较。

## 快速示例


```python
score = 85
print(score >= 70)   # True
print(score == 100)  # False

grade_letter = "B"
print(grade_letter == "B")  # True
print(grade_letter != "A")  # True
```

比较运算符总是给出布尔值。熟练掌握这些运算符后，就可以在下一页使用**逻辑运算符**（`and`、`or`、`not`）组合条件。
