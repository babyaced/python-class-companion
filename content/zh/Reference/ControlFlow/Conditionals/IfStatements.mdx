# `if` 语句

您将学习如何仅在**条件**为真时运行代码块。您可以使用 `if`，以及可选的 `else` 和 `elif` 来做到这一点。条件是一个非真即假的表达式（例如，`年龄 >= 13`）。您将在比较运算符和逻辑运算符页面看到更多编写条件的方法。

## 当条件为真时运行代码

有时，你希望程序在一种情况下执行一组步骤，而在另一种情况下执行另一组步骤。例如：在一个电影应用程序中，当用户年龄在 13 岁以下时，推荐适合家庭观看的电影，而当用户年龄在 13 岁或以上时，则推荐范围更广的电影。

在 Python 中，您需要使用关键字 `if`，然后是条件，最后是冒号。当条件为真时运行的代码在下面缩进。


```python
age = 15

if age < 13:
    # recommend family-friendly movies
if age >= 13:
    # recommend a wider range of movies
```

程序会检查 `age` <GlossaryTerm term="变量">变量</GlossaryTerm>的值。如果 `age < 13` 为 true，则运行第一个程序块。如果 `age >= 13` 为 true，则运行第二个程序块。此处 `age` 为 15，因此跳过第一个程序块，运行第二个程序块。

我们可以通过使用 `else` 来更清楚地说明这一点，并避免重复检查同一个想法。

## `else`: 当条件为假时

`else` 表示 "当 `if` 条件为假时执行此操作"。在 `else` 后不需要写一个新的条件--你只是在说 "否则，运行此代码块"。

在电影示例中，如果用户不小于 13 岁，那么他们必须是 13 岁或以上。因此我们只需检查一次：


```python
age = 15
if age < 13:
    # recommend family-friendly movies
else:
    # recommend a wider range of movies
```

如果 `age < 13` 为真，我们将运行第一个程序块。否则，我们运行 "else "代码块。一次检查，两种结果。

## `elif`: 超过两种情况

当有三个或更多情况时，使用 `elif` ("else if "的缩写)。Python 会先检查 `if`，然后依次检查每个 `elif` ，如果都不为真，则检查 `else`。只要有一个条件为真，就运行该代码块，跳过其他代码块。

例如：针对 13 岁以下、13-17 岁和 18 岁以上的不同建议：


```python
age = 15
if age < 13:
    # recommend family-friendly movies
elif age < 18:
    # recommend teen movies
else:
    # recommend adult movies
```

Python 首先检查 `age < 13`。如果为假，则此人是 13 岁或以上，因此 `elif age < 18` 能正确捕获 13-17 岁的人。其他人（18 岁以上）则使用 `else`。条件的顺序起作用。

**请记住：** `elif`只有在前一个`if`（以及任何前一个`elif`）为假时才会被检查。所以你选择的是一个分支。

## `if` 与 `elif`: 何时使用哪个

`if` 总是进行检查。而 `elif` 只有在上面所有条件都为假时才会执行检查。因此，不能总是交换使用。

例如：我们想祝贺一名学生全勤**和**上了光荣榜。如果我们在第二条信息中使用 `elif` ，则只打印一条信息：


```python
student_has_perfect_attendance = True
student_on_honor_roll = True

if student_has_perfect_attendance:
    print("Congrats on perfect attendance!")
elif student_on_honor_roll:
    print("Congrats on making the honor roll!")

# Output: Congrats on perfect attendance!
# (Honor roll message is skipped—elif only runs when the first condition was false.)
```

由于第一个条件为真，Python 会运行该代码块并跳过 `elif`。要在两个条件都为真时打印两条信息，请使用两个单独的 `if` 语句：


```python
student_has_perfect_attendance = True
student_on_honor_roll = True

if student_has_perfect_attendance:
    print("Congrats on perfect attendance!")
if student_on_honor_roll:
    print("Congrats on making the honor roll!")

# Output:
# Congrats on perfect attendance!
# Congrats on making the honor roll!
```

每个 "if "都是单独检查的，因此两个区块都能运行。

## 摘要

- 使用 `if` 仅在条件为真时运行代码。
- 使用 `else` 表示 "否则"，而 `elif` 则用于有几种情况并想选择其中一种时。
- 条件是一个真或假的表达式。在**比较运算符**和**逻辑运算符**页面，您将看到如何编写和组合条件。在**循环**中也会用到同样的思想，即多次运行一个代码块。
