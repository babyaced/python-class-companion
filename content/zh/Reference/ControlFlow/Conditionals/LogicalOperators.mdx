# 逻辑操作符

本节将学习**逻辑运算符**--它们可以让你将两个或多个条件合二为一。您已经使用过比较运算符来构建单个条件（例如，"年龄 >= 18"）。逻辑运算符可以让你说 "这个条件**和**那个条件 "或 "这个条件**或**那个条件 "或 "**不**这个条件"。

Python 中的三个逻辑运算符：

- **`and`** - 只有当两个条件都为真时才为真。
- **`or`** - 至少一个条件为真时为真。
- **`not`** - 翻转结果："真 "变为 "假"，"假 "变为 "真"。

## `and` 运算符

当需要两个**条件都为真时，使用 `and` 运算符。如果其中一个为假，则整个表达式为假。


```python
age = 20
has_id = True

if age >= 18 and has_id:
    print("You can enter the club!")
else:
    print("Sorry, you cannot enter the club.")
```

只有年满 18 岁且***有身份证件的人才能参赛。这两项必须都是真实的。

## `or` 运算符

当**个条件中至少有一个**为真时，使用 `or` 运算符。如果其中一个条件为真，则整个表达式为真。


```python
day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")
else:
    print("It's a weekday.")
```

如果当天是周六或周日，我们将打印周末信息。只有一个条件必须为真。

## `not` 运算符

使用 `not` 可以翻转条件：如果条件为真，则 `not` 使其为假，反之亦然。


```python
is_raining = False

if not is_raining:
    print("It's a nice day outside!")
else:
    print("Don't forget your umbrella!")
```

这里，`is_raining`是`假`。所以 `not is_raining` 是 `True`，我们就会打印 "外面的天气真好！"。如果 `is_raining` 是 `True`，我们就会打印雨伞信息。

## 组合逻辑运算符

在一个条件中可以使用多个逻辑运算符。括号有助于明确顺序。


```python
age = 25
has_id = True
is_member = False

if (age >= 18 and has_id) or is_member:
    print("You can enter the club!")
else:
    print("Sorry, you cannot enter the club.")
```

如果（您年满 18 周岁并有身份证）**或**您是会员，则可以输入。括号表示我们首先检查 "18 岁以上并有身份证"，然后才允许使用 "或 "输入会员。

<Callout type="info" title="运算顺序">
Python 按此顺序评估逻辑运算符：先是 `not`，然后是 `and`，最后是 `or`。使用括号（如上面的示例）可以明确您的意图，并帮助您避免错误。
</Callout>
