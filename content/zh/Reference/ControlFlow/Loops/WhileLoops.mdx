# While Loops

A `while` loop runs a block of code **as long as a condition is true**. Unlike a `for` loop, you don't iterate over a sequence — you keep going until something changes.

## Basic syntax

```python
while condition:
    # code to run while condition is true
```

The condition is checked before each iteration. When it's `False`, the loop stops and the program continues after it.

## Basic example

```python
count = 1
while count <= 5:
    print(count)
    count += 1
# Output: 1 2 3 4 5
```

Each iteration:
1. Check: is `count <= 5`?
2. If yes, print `count` and add 1
3. When `count` reaches 6, the condition is false — loop ends

<Callout type="warning" title="Always update your condition">
If nothing inside the loop changes the condition, it will never become `False` and your program will run forever. Make sure something inside the loop moves toward ending it.
</Callout>

## Infinite loops

A loop that never stops is called an **infinite loop**. This usually happens by accident:

```python
# Infinite loop — count never changes!
count = 1
while count <= 5:
    print(count)
```

If your program freezes, an infinite loop is often why. Press `Ctrl + C` in the terminal to stop it.

Sometimes infinite loops are intentional — a game loop or a program waiting for input — but you should always have a way to break out of them.

## Validating user input

`while` loops are great for repeating a prompt until the user gives a valid answer:

```python
age = int(input("Enter your age: "))
while age < 0:
    print("Age can't be negative.")
    age = int(input("Enter your age: "))

print(f"Your age is {age}.")
```

循环会一直询问，直到 `age` 大于等于 0。

### 在特定输入时停止

您也可以让用户通过输入特定值来退出循环。使用 `break` 立即停止循环：

```python
while True:
    answer = input("Enter a number (or 'q' to quit): ")
    if answer == "q":
        break
    print("You entered:", answer)

print("Goodbye!")
```

`while True` 创建一个有意的无限循环——只有在触发 `break` 时才会结束。当您希望程序持续运行直到用户决定停止时，这种模式很常见。

<Callout type="info">
  `break` 会立即退出循环，跳过当前迭代的剩余部分和条件检查。程序将继续执行循环之后的内容。
</Callout>

## 何时使用 `while` 循环

当您**事先不知道需要重复多少次**时，使用 `while` 循环——您只需知道应该继续执行的条件。如果您在遍历列表或固定范围，`for` 循环通常更简洁。

| 情况 | 使用的循环 |
|-----------|-------------|
| 循环固定次数 | `for` 与 `range()` |
| 遍历列表中的每个元素 | `for` |
| 循环直到条件改变 | `while` |
| 持续询问直到获得有效输入 | `while` |

## 摘要

- `while` 循环在其条件保持为真时重复执行。
- 确保循环内部有某些内容发生变化，使条件最终能变为假。
- 当您不知道需要重复多少次时，使用 `while`。
