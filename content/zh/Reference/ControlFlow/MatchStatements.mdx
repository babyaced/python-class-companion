# 匹配语句

Python 中的 `match` 语句允许您将一个值与一系列模式进行比较，并根据哪种模式匹配来执行代码。

下面是一个如何使用 `match` 和 `case` 语句的简单示例：

假设我们想根据星期打印一条消息：


```python
day = "Monday"

match day:
    case "Monday":
        print("Start of the work week!")
    case "Wednesday":
        print("We're halfway through the week!")
    case "Friday":
        print("It's almost the weekend!")
    case _:
        print("Just another day.")
```

在本例中，"match "语句针对每个 "case "检查 "day"<GlossaryTerm>变量</GlossaryTerm>的值。

- 如果 `day` 是 `"星期一"，则打印 "工作周开始！"
- 如果 `day` 是 `"星期三"，则打印 "本周已经过半！"
- 如果 `day` 是 `"星期五"，则打印 "快到周末了！"
- `_` 的情况是一个 "通配符"，可以匹配前面的情况所不包括的任何内容，因此对于任何其他的 `day` 值，它都会打印 "又是一天"。

从上面的解释中可以看出，"match "语句类似于多个 "if"、"elif "和 "else "语句，但它更易读、更简洁。

在编写代码的过程中，你会发现在某些情况下，使用 `match` 和 `case` 会比使用多个条件语句使代码更清晰、
但一般来说，当一个变量可以有许多不同的值，而你又想对每个值进行不同的处理时，`match` 就特别有用了。
