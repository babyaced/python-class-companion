# 输出格式化

在 Python 中显示输出时，您可能希望将其格式化，以便于阅读。
Python 提供了几种格式化输出字符串的方法。下面是一些常用的方法：

## 使用 f-string (Python 3.6+)

f-string 提供了一种使用大括号 `{}` 将表达式嵌入字符串字面量的方法。它们的前缀是字母 `f`。


```python
# Example of f-string formatting
name = "Alice"
age = 30
print(f"Name: {name}, Age: {age}")
```

这将输出


```
Name: Alice, Age: 30
```

您可以直接在大括号内使用变量名，Python 会用它们的值替换它们。
不过，这只有在字符串前加上 `f` 才有效，如上所示。

## 使用 `format()` 方法
使用 `format()` 方法可以格式化字符串，方法是在字符串中放置大括号 `{}` 作为占位符，然后在字符串上调用 `format()` 方法。


```python
# Example of format() method
name = "Bob"
age = 25
print("Name: {}, Age: {}".format(name, age))
```

这里，第一个 `{}` 由格式化方法的第一个参数（`name`）代替，第二个 `{}` 由格式化方法的第二个参数（`age`）代替。

这将输出


```
Name: Bob, Age: 25
```

## 使用 `%` 操作符
操作符 `%` 也可用于字符串格式化。
它使用格式指定符来指示插入数值的位置。


```python
# Example of % operator formatting
name = "Charlie"
age = 28
print("Name: %s, Age: %d" % (name, age))
```

这里的占位符是 `%s` 和 `%d` 。
要插入的值位于 `%` 操作符后的括号中。它们按出现的顺序插入。


这将输出


```
Name: Charlie, Age: 28
```

你可能想知道为什么我们在占位符中使用字母 `s` 和 `d`。

- `%s` 用于字符串。
- `%d` 用于整数（十进制数）。
- `%f` 用于浮点数。

这就意味着你可以适当地格式化不同类型的数据。
您必须确保针对插入的数据类型使用正确的格式指定符。

## 摘要

我们已经介绍了在 Python 中格式化输出的三种常用方法：f-string、`format()` 方法和 `%` 操作符。
