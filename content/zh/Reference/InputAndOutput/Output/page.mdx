# 打印输出

在 Python 中，可以使用 `print()` 函数将输出打印到控制台。

它是 Python 中最常用的函数之一，在本课中也会经常用到。

## 基本打印

要打印一条简单的信息，您可以像这样使用 `print()` 函数：


```python
# Example of basic printing
print("Hello, World!")
```

我们要打印的*字符串*需要放在括号内，并用引号括起来。

运行这段代码后，将输出


```
Hello, World!
```

不过，我们并不局限于打印字符串。我们还可以打印数字和其他数据类型。例如


```python
# Printing different data types
print(42)            # Printing an integer
print(3.14)         # Printing a float
print(True)         # Printing a boolean
```

这将输出


```
42
3.14
True
```

## 打印变量

要在 Python 中打印变量，只需将变量名传递给 `print()` 函数。例如


```python
# Printing variables
name = "Alice"
age = 30
print(name)
print(age)
```

这将输出


```
Alice
30
```

在接下来的章节中，你将看到可以在打印语句中将字符串和变量组合在一起，实现更复杂的输出。

## 打印多个项目

你还可以在 `print()` 函数中用逗号分隔多个项目，一次打印多个项目。例如


```python
# Printing multiple items
name = "Alice"
age = 30
print("Name:", name, "Age:", age)
```

在这里，我们要打印一个字符串，然后是变量 `name` 的值，接着是另一个字符串，然后是变量 `age` 的值。

这将输出


```
Name: Alice Age: 30
```

打印多个项目时，Python 会自动在每个项目之间添加一个空格。您可以把逗号想象成输出中的空格。
