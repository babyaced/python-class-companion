# 处理文件

有时，你希望程序在关闭后仍能记住数据，比如用户添加的分数或条目列表。这就是文件的作用。Python 使从文件读取数据和向文件写入数据变得很容易。

## 打开文件

使用 `open()` <GlossaryTerm term="function"> 函数</GlossaryTerm> 打开文件。您可以向它传递文件名和一个 **mode** 来告诉 Python 您想对它做什么：

| 模式 | 它的作用 | 文件名
|------|-------------|
| `"r"` | 读取 - 打开一个现有文件以读取其中的内容。
| `"w"` | 写--创建或覆盖文件
| `"a"` | 追加 - 添加到现有文件的末尾（或创建文件） | `"b"` | 添加到现有文件的末尾（或创建文件


```python
my_file = open("scores.txt", "r")
```

这会以读取模式打开 `scores.txt`，并将结果存储在 <GlossaryTerm> 变量</GlossaryTerm>中。您将使用该变量进行读取或写入。

## 从文件中读取

最常见的读取方式是逐行循环读取文件：


```python
my_file = open("scores.txt", "r")
for line in my_file:
    print(line.strip())
my_file.close()
```

每通过一次循环，就会得到下一行字符串。.strip()`会去掉字符串末尾的换行符--你几乎总是需要它。

如果你需要一次性将整个文件转换成一个字符串，可以使用 `read()`：


```python
my_file = open("scores.txt", "r")
content = my_file.read()
print(content)
my_file.close()
```

如果想要以列表形式显示所有行，请使用 `readlines()`：


```python
my_file = open("scores.txt", "r")
lines = my_file.readlines()  # returns ["line 1\n", "line 2\n", ...]
my_file.close()
```

对于大多数项目来说，"for line in file "循环就足够了。

## 写入文件

以写入模式打开并使用 `write()` <GlossaryTerm term="method">方法</GlossaryTerm>：


```python
my_file = open("scores.txt", "w")
my_file.write("Alice: 95\n")
my_file.write("Bob: 88\n")
my_file.close()
```

<Callout type="warning" title="写模式会替换文件">用 "w "打开文件时，会清除文件中的所有内容。
使用 `"w"`打开文件时，会删除文件中已有的内容。如果想在文件末尾添加内容而不删除原有内容，请使用 `"a"` （追加）。
</Callout> 呼出


```python
my_file = open("scores.txt", "a")
my_file.write("Carol: 92\n")
my_file.close()
```

## 关闭文件

使用 `close()`完成后，一定要关闭文件。这样可以将所有内容保存到磁盘，并释放资源--就像保存和关闭文档一样。


```python
my_file.close()
```

## 使用 `with`（推荐）

使用 `with` 代码块处理文件是一种更简洁的方法。它能在代码块结束时自动关闭文件，即使出了差错也是如此：


```python
with open("scores.txt", "r") as my_file:
    for line in my_file:
        print(line.strip())
```

您不需要自己调用 `close()`。一旦缩进块结束，Python 就会处理它。这是实际代码中最常见的模式。
