# 用户输入

用户输入是程序运行时接收用户数据的一种方式。
在 Python 中，我们可以使用**内置的** `input()`函数来获取用户输入。

要使用`input()`函数，我们只需调用它，并选择性地*pass*一个字符串作为提示显示给用户。

让我们询问用户的姓名，然后向他们问好：


```python
name = input("What is your name? ")
print("Hello, " + name + "!")
```

代码的工作原理如下
1.我们调用`input()`函数，提示`"你叫什么名字？"`
2.程序等待用户输入姓名并按 Enter 键。
3.用户输入的内容将存储在变量 `name` 中，因为输入函数总是返回输入的字符串。
4.最后，我们会打印一条包含用户姓名的问候信息。

通过类型转换，您还可以使用 `input()` 函数从用户获取不同类型的输入。

例如，如果想从用户那里获取一个数字，可以使用 `int()` 函数将输入字符串转换为整数：


```python
ageString = (input("How old are you? ")) # use the input function to get user input and store in ageString variable
age = int(ageString) # convert our ageString variable that is string type to integer type
print("You are " + str(age) + " years old.") # print the message to the user - converting age back to string for concatenation
```
