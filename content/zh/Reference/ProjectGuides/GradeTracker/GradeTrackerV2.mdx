# Project 2 (v2): Grade Tracker — By Subject

You'll build a **continuation** of the Grade Tracker that stores grades **by subject** in a **list of lists**. Each inner list holds the subject name and that subject's grades. You'll still use lists, loops, conditionals, and file I/O—same topics as the original Grade Tracker, with a 2D data structure.

<Callout type="warning" title="Before you start!">
Keep the original Grade Tracker code in a separate file!
This will be a separate project that you can work on each day if you finish the steps for each day of the original Grade Tracker.
</Callout>

**Data shape:** One list; each item is a list of the form `[subject, grade1, grade2, ...]`. For example: `[["Math", 85, 92, 78], ["Science", 88, 91], ["English", 76, 82, 90]]`. Index `0` of each inner list is the subject (a string); the rest are that subject's grades (numbers).

---

## Phase 1 (Day 1): Lists and Loops

### Learning focus
Create a list of lists, loop through the outer list and each inner list to display grades and compute sums.

### Tasks

<Checklist>
  <Checkbox id="proj2v2-phase1-task1" label="Start with a static list of lists, e.g. grades_by_subject = [['Math', 85, 92, 78], ['Science', 88, 91], ['English', 76, 82]]. This is your data for the rest of the tasks." />
  <Checkbox id="proj2v2-phase1-task2" label="Use a for loop over the outer list. For each inner list, index 0 is the subject; the rest are grades. Print the subject, then print each grade." />
  <Checkbox id="proj2v2-phase1-task3" label="For each subject's list, loop over the grades and add them up. Print each subject and its sum." />
  <Checkbox id="proj2v2-phase1-task4" label="For each subject, compute average = sum ÷ number of grades. Print each subject and its average." />
</Checklist>

**Topics covered:** [Lists](../../Variables/TypesOfVariables/DataStructures/Lists), [Loops](../../ControlFlow/Loops)

**End of Day 1:** You have a program with a list of lists that prints each subject's grades, sum, and average.

### Challenge (optional)
Add a new subject and its grades to the list (e.g. prompt for subject name and a few grades with `input()`), or use a loop to add several grades to one subject.

### Tasks

## Phase 3 (Day 3): File I/O

### Learning focus
Save the list of lists to a file and load it back so data persists between runs.

### Before you start
Decide a simple file format. One option: each line is one subject, with subject name and grades separated by commas, e.g. `Math,85,92,78` then `Science,88,91`. Create a file (e.g. `grades.txt`) in the same folder as your program.

### Tasks

<Checklist>
  <Checkbox id="proj2v2-phase3-task1" label="Write a function that opens a file for writing. Loop over your list of lists; for each inner list, write one line with subject and grades joined by commas." />
  <Checkbox id="proj2v2-phase3-task2" label="Write a function that opens the file, reads each line, and splits by comma. The first part is the subject; the rest are grades (convert to float). Return a list of lists." />
  <Checkbox id="proj2v2-phase3-task3" label="When the program starts, load the list of lists from the file. At the end, save it back so data persists between runs." />
</Checklist>

**Topics covered:** [Handling files](../../InputAndOutput/HandlingFiles), [Lists](../../Variables/TypesOfVariables/DataStructures/Lists), [Loops](../../ControlFlow/Loops)

**End of Day 3:** You have a full grade tracker by subject: load from file, show each subject's stats, and save back so data persists.
