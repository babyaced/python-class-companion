# Project 1: Environmental Tracker

For this project, you'll create a simple Python program to track your environmental impact, focusing on the water used for showers. You'll get introduced to basic Python concepts like input/output, variables, constants, and simple calculations.

---

## Stage 1: Input and Output

### Learning focus
Get comfortable with capturing user input and displaying output.

### Tasks

<Checklist>
  <Checkbox id="proj1-stage1-task1" label="Write code that prompts the user to enter the number of showers they take in a week and store this value in a variable." />
  <Checkbox id="proj1-stage1-task2" label="Write code that prompts the user to enter how long their average shower lasts (in minutes) and store this value in another variable." />
  <Checkbox id="proj1-stage1-task3" label="Display the total number of showers and average shower duration on the screen for the user." />
</Checklist>

**Topics covered:** [User Input](../../InputAndOutput/UserInput), [Output](../../InputAndOutput/Output)

### Tasks
