# Pillow 活动

在本活动中，你将探索用于图像处理的 Pillow 库——加载图像、应用变换、阅读文档并保存结果。

**首先安装 Pillow**（如果尚未安装）：

```bash
pip install pillow
```

然后在文件顶部导入所需内容：

```python
from PIL import Image, ImageFilter
```
