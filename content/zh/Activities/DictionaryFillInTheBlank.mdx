# 字典填空

通过填写缺失的键和值来练习构建字典。

---

## 第 1 部分 — 填写空白

下面的字典缺少一些键和值。用正确的答案填写每个 `____`。不知道的内容可以 Google。

```python
movie = {
    "title": "The Dark Knight",
    "____": 2008,
    "genre": "____",
    "____": "Christopher Nolan",
    "cast": ["Christian Bale", "____", "Aaron Eckhart"]
}
```

<Checklist>
  <Checkbox id="fitb-1" label="填写 2008 的缺失键（提示：这个数字代表什么？）。" />
  <Checkbox id="fitb-2" label="填写类型（genre）。" />
  <Checkbox id="fitb-3" label='填写 "Christopher Nolan" 的缺失键。' />
  <Checkbox id="fitb-4" label="填写缺失的演员（提示：他凭借这个角色获得了奥斯卡奖）。" />
</Checklist>

**涉及主题：** [什么是字典？](/zh/Reference/Variables/TypesOfVariables/DataStructures/Dictionaries/page)

## 第 2 部分 — 自己创建

现在为你选择的电影创建一个字典。使用相同的字段。

```python
movie = {
    "title": "",
    "year": ____,
    "genre": "",
    "director": "",
    "cast": ["", "", ""]
}
```
