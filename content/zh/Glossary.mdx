使用本页查找您在课程中看到的术语。如果有些术语听起来很陌生，请在这里查看快速定义。

## 关键词

Python 保留给自己使用的词。不能用作变量名、函数名等。例如`if`, `else`, `while`, `for`, `def`, `class`。

## 标识符

您在代码中为变量、函数或类赋予的名称。它必须遵守规则（例如，以字母或下划线开头，不能有空格），并且不能与关键字相同。

## 变量

表示一个值的名称。您可以在程序运行时更改该值。例如，"score = 0 "创建了一个名为 "score "的变量，并赋予其值 "0"。

## 函数

执行特定任务的可重用代码块。函数可以接收输入<GlossaryTerm>参数</GlossaryTerm>和<GlossaryTerm>返回</GlossaryTerm>输出值。

### 类

构建对象的蓝图。它描述了这些对象所持有的数据（属性）以及它们能做的事情（方法）。当您学习更高级的 Python 时，就会用到类。

## 对象

程序中拥有数据并能执行操作（方法）的值。对象通常由类创建；"object "是其中一个类的统称。

### 实例

由类创建的特定对象。对于类中定义的属性，每个实例都可以有自己独特的值。我们通常使用 "实例 "一词来指代一个对象，以强调它是一个类的具体实例。

### 属性

属于对象或类的变量。属性包含描述对象或类的数据。

### 方法

属于对象或类的函数。

## 可变

创建后可以更改的东西。在 Python 中，列表和字典是可变的--您可以添加、删除或更改项。

## 不可变的

创建后无法更改的东西。在 Python 中，字符串和整数是不可变的。如果您 "更改 "了一个字符串，您实际上是创建了一个新的字符串。

### 语法

如何写代码才能让 Python 读懂的规则。漏掉了冒号或括号？这就是语法错误。

## 语义

你的代码*做了什么*--它的含义和行为。如果语法正确，但程序行为错误，那就是语义（逻辑）问题。

## 执行

运行您的代码，使其真正执行您所写的内容。运行 Python 程序时，解释器会逐行读取并执行您的代码。
