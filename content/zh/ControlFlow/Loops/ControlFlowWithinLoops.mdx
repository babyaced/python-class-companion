# 循环内的控制流

本节将学习如何在循环内部使用 `if`、`break` 和`continue`。在循环内部设置条件语句可以让你在不同的迭代中执行不同的工作--例如，只打印偶数或在找到一个值后立即停止。你还会看到两条只有在循环内部才有意义的语句：break"（立即退出循环）和 "continue"（跳到下一次迭代）。

## 在循环中使用 `if

你可以在循环内部使用 `if`（以及 `elif` 和 `else`）。每次迭代都会对条件进行检查，因此可以对每个项目做出不同的反应。

示例：只打印从 1 到 10 的偶数。我们对数字进行循环，只有当数字是偶数（能被 2 整除且没有余数）时才打印：


```python
for number in range(1, 11):
    if number % 2 == 0:
        print(number)
```

每次循环时，我们都要检查条件。为真时，我们打印。如果为假，我们就什么也不做，然后转到下一个数字。输出：2, 4, 6, 8, 10.

一般来说，你可以在循环中使用你所学过的任何控制流--"if"、"elif"、"else"。循环和条件式共同表达 "对于每个项目，只有当该条件为真时，才执行该操作"。

## `break` 语句

`break` 语句立即退出循环。不再迭代运行。我们通常在找到我们要找的东西或某个条件说 "停止 "时使用它。

例如：搜索数字 7，找到后立即停止：


```python
number_we_are_looking_for = 7
current_number = 1
while current_number <= 10:
    if current_number == number_we_are_looking_for:
        print(f"Found the number: {current_number}")
        break
    print(f"Checking number: {current_number}")
    current_number += 1
```

当 `current_number` 为 7 时，我们打印信息并点击 `break`。循环就此结束；我们不会检查 8、9 或 10。如果没有`break`，循环会一直进行到 10。

## `continue` 语句

`continue` 跳过当前迭代的剩余部分，跳到下一个迭代。循环继续运行，我们只是跳过一次。当我们想忽略某些项目但继续处理其余项目时，就会用到它。

例如：使用 `continue` 跳过奇数，只打印偶数：


```python
for number in range(1, 11):
    if number % 2 != 0:
        continue
    print(number)
```

当数字是奇数时，我们点击 "继续"，跳过 "打印"。当数字是偶数时，我们不点击 "继续"，因此运行 "打印"。结果：2、4、6、8、10。输出结果与第一个示例相同，但这次我们使用了 `continue` 来跳过奇数，而不是在打印周围使用 `if`。

**简而言之：** `break` 表示 "退出循环"。`continue` 表示 "跳过本次迭代，进入下一次迭代"。

## 摘要

你可以在循环中使用 `if`、`elif` 和 `else` 来运行代码，只有当条件在该迭代中为真时才运行代码。`break` 提前退出循环；`continue` 跳到下一次迭代。通过它们的组合，您可以构建对条件做出反应的循环，并在需要时停止或跳转。
